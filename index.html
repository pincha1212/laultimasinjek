Proyecto 2

Simulador de tienda en linea(Beta)

La tienda en línea es un sitio web que ofrece productos y servicios a los clientes de forma virtual, permitiendo la compra desde cualquier lugar con acceso a internet. El sitio web cuenta con una interfaz intuitiva y fácil de usar, además de un sistema de pago seguro y opciones de envío y devolución para la comodidad del cliente. Todo esto se realiza con el objetivo de proporcionar una experiencia de compra satisfactoria y conveniente para el usuario.
